推荐一些 macOS 下众所周知的应用

Homebrew

虽然这家伙并不是应用，但我就想把它放在最前面说
可以使用它来安装一些第三方包和一些 macOS 的软件

安装，升级，卸载都将是一条命令都事儿

官网：查看链接

一些基础命令：

brew (cask) install xxx | 安装 xxx，部分应用或包需要加 cask
brew (cask) uninstall xxx | 卸载 xxx，你安装有 cask 卸载就有
brew (cask) list ｜ 显示你已经安装了啥
brew update (xxx) ｜ 升级(xxx)
brew cleanup ｜ 清理
brew doctor | 诊断

HandBrake

一个纯净、免费的视频格式转换软件

支持调整视频分辨率、音轨、帧率、码率等。。。

IINA

欢迎界面：

视频倍速最低可调至 0.25x，最高可调至 16x

还可在线查找字幕的说

GitHub 开源项目:查看链接

XLD

可以将音乐在各大格式之间转换，抓取 CD（虽然我没有用过）
转为 .m4a 后导入 iTunes 即可在 iPod 上享受到“无损音质”

支持格式：WAV、AIFF、PCM、Wave64、MEPG-4 ACC、Apple Lossless、Direct Stream Digital、FLAC、MP3、Opus、Ogg 等格式

官网：查看链接

LICEcap

一款录制屏幕为 GIF 的软件，可调帧率，录制过程中可随意调节录屏范围

官网：查看链接

uTools

以聚焦搜索式的方式带来各种自定义插件，将不同的工具集合起来，智能的处理方式，提升效率不错哒
