正经儿写

材料：已 root 手机一部（目标高通处理器的手机）

MT 管理器（我习惯用而已，其它的文件管理器也行
Poweramp 软件

理论大部分手机都支持，不试一试怎么知道呢？

什么？你要外放音质好？算了吧，捡洋垃圾去

需要 root 部分（无 root 可以跳过）（针对高通处理器部分）（可能大部分人知道，但我还是要写）

按路径查看是否有此录目
/sys/module/snd_soc_wcd9330/
/sys/module/snd_soc_msm8x16_wcd/
有的话，选有的哪个即可
命令：
echo 1 > /sys/module/snd_soc_wcd9330/parameters/high_perf_mode
echo 1 > /sys/module/snd_soc_msm8x16_wcd/parameters/high_perf_mode

意思是开启 Hi-Dac（有玄学提升
一般重启失效，想开机自启的话，可以在 /system/etc/init.qcom.post 的最后一行加入
注，Android8.0 以上路径可能是 /system/vender/etc/init（手上没有 Android8+已 root 的机子，如有错误，欢迎指出修改

开启抗阻检测

命令：
echo 1 > /sys/module/snd_soc_wcd9xxx/parameters/impedance_detect_en

开机自启同上

poweramp“优化”

Poweramp→ 设置 → 音频 → 输出 →OpenSL ES Hi-Res 输出 → 开启开关
Poweramp→ 设置 → 音频 → 重采样 →Sox 重采样
若开启 Hi-Res 后无声音，说明设备不支持，关闭即可

（另外，EMUI 卸载自带音乐应用后将无法使用 Hi-Res 输出，冻结则不影响）
