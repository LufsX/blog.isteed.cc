申请条件

  • 无侵入性广告
    • 国内部分「免费空间」与「免费主机」搭建的站点，不接受申请。
  • 网站域名为 Public Suffix List 中的子域
  • 非采集、无 任何 原创内容的站点
    • 内容不追求多、质量好，但要符合大众道德观念
    • 单独的资源站不接受，抱歉
  • 请注意长期无法访问或未添加本站为友链的站点可能将被移除，若更换域名，请及时更新资料
  • 最终解释权归 水怪 (LufsX) 所有 🤣

添加本站～

  • 将本站添加至自己的友链页面中
    • 站点名称：Lufs's Blog 或 LufsX
      • 非强制，合适即可
    • URL：https://blog.isteed.cc/ 或 https://isteed.cc
      • 与上面两个名称所对应
    • 简介：一个假的博客~
      • 非强制，可随兴发挥～
    • Slogan：
      • Life feeds on negative entropy.
      • 非强制，简介或 Slogan 自行选择～
    • Logo：
      • Avatar：
        • 36x36 png / webp
        • 72x72 png / webp
        • 128x128 png / webp
        • 256x256 png / webp
        • 512x512 png / webp
      • Favicon：
        • 16x16 png / webp
        • 32x32 png / webp
        • 180x180 png / webp
        • 192x192 png / webp
        • svg
      • 顺带一提，图标、头像都有 Webp 版本，网址改个后缀就可以了
      • 自行选择合适的大小哈～

准备资料～

  • 准备一个自己想展示的站点图片（Logo）
    • 长度与宽度应小于 512px
    • 使用 webp、png、jpg、svg、ico 等常见文件格式哈～
    • 使用链接形式，尽量保证可以持久访问（HTTPS ONLY!）
    • 尽量不要 NSFW（Not Safe/Suitable For Work）
  • 准备需要展示的友链名称
    • 尽量不要 NSFW
  • 准备一条 Slogan
    • 尽量不要 NSFW
