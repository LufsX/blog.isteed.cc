迷你主机的需求可能对我来说是个伪命题……

以轻便整洁而称，但后面就发现一套带下来不比笔记本轻松，线材也没多整齐（

但说实在的，虽然性价比没有自组 ITX 这类的高，但也还算不错的

注：由于机械革命无界 S MINI 就是铭凡 UM780 PRO 的贴牌版本，故本文中所有描述均适用于铭凡 UM780 PRO，毕竟唯一的不同就是机子表面 LOGO 图案不一样

缘起

原来我是 MacBook Air 用过来的，用惯了 macOS，就索性搞了台 M1 的 MacBook Air 用

但在宿舍没事儿干能干啥，哪肯定是臭打游戏啦～

就有了个搞台 Windows 电脑的想法

就开始陆陆续续的蹲个好价攒点配件，固态、显示器、内存、迷你主机、便携屏，跨度有足足的一年

有些地方有些太无谓的奢侈了，没法子，被果子天价内存硬盘憋坏了

售后

我这个机子其实出过四次问题，且听我娓娓道来

先叠甲，仅代表个人体验，以及以下问题仅针对首发机子/维修机子

第一次（2023-12-19）是主板的设计问题，前面天气热可能没出现，后面冷了就出现了，毕竟哪段时间刚好寒流来了（快递来回五天，维修半天，换了个主板）
第二次（2023-12-26） 是换完主板后，但机子液金手工涂的，机子打游戏动不动就上 90°C（快递来回五天，维修半天，这次直接主板和散热一起换）
第三次（2024-01-08）是游戏打着打着黑屏闪退，重装系统，拷机拷内存都没问题（快递来回四天，直接换货）
第四次（2024-03-04）是底部小风扇异响，声音极大，又是送售后了（快递来回五天，维修半天，换了底部风扇）

所以最后我是首发的机子（八月产）变成了换货的备件机（十二月产），售后态度还是很不错的，就是快递时间太折磨人（要不是双向包邮，不然难顶）

截止目前也没其它问题了，网卡从原来首发的 RZ608 换成了 AX210，现在拷机温度最高也就 75°C（冬天加持～）

性能

有闻 AMD 核显很强，且相较于同定位上代的 6800H 相比，CPU GPU 性能都有进步，曾经推荐朋友买过 6800H 的笔记本，在其上体验过部分游戏，均是能玩水平，而且制程是神的台积电，能效高些

游戏在能玩水平已经很能够满足（快递来回五天，维修半天，换了个主板） macOS 用户了（我尝试过 M1 的 MacBook Air 使用 Rosetta 2 转译 x86-64，使用 wine 做到兼容 Windows 层，使用 DXVK 转译 DX11 图形 API 来运行战地 1……（这又是另外一个故事了）

7840H/HS 简单过一下参数，8C16H 纯大核，全核 4.7GHz，单核 5.1GHz，780M 核显

跑分也简单过下 R23 单 1792 多 17141，CPUZ 单 654.3 多 6950.8，大抵就是桌面的 13400 以上 13500 以下，3D MARK 没买没有

（测试环境基于 BIOS 设置性能模式 65W 功耗，内存完整性保护关闭，Meltdown 与 Spectre 保护禁用，虚拟化 SVM 开启，Hyper-V 开启）

这个机子用的液金散热，散热真的猛，按冬天的情况来说，游戏/拷机最高温度不超过 78°C，全程维持 65W 的 APU 功耗（但也因为液金，手涂不得行）

硬盘温度方面很低，毕竟底部风扇直接给硬盘散热，有着散热片贴着，硬盘温度最高也就六十来度

体验

装内存硬盘需要很不优雅的拔掉底下的四个脚垫（怪不得给你多弄两个新脚垫），拧下螺丝，翘棒翘起，底板上连接着风扇和天线

散热器上有两条给硬盘用的蓝色保护膜的散热垫

机子呈天包地的结构，金属外壳，无线网络在高负载时贼差劲，需要把机子立起来才能畅玩

底下的哪条内存可以装散热鳍片，防止内存叠叠乐

这台机子是真的小，巴掌大的主机，性能不弱，放哪儿都行……看向了我宿舍哪块贼大的桌子（陷入了沉思

外围接口量大管饱，有下面这些

口子数量够多，但是把两 C 口放正面是要干甚？换个到后面去多美好，虽说铭凡的 UM780 XTX 改了，但厚了，还贵了

这个大小口袋都装的下去，确实「便携」，如果不考虑要带电源/屏幕/键盘/鼠标的话

软件

机子本身就是铭凡贴牌的机子，BIOS 和 UM790 PRO 的完全通用，刷完后 APU 甚至 7840H 荣升 7840HS

但铭凡的 LOGO 不好看，我还是刷回了机械革命的 BIOS

游戏

简要总结，1080P 低画质基本绝大部分游戏畅玩

以下体验基于 AMD 驱动版本 24.3.1

原神：1080P 中画质，须弥城 55 ～ 60FPS
崩坏：星穹铁道：1080P 非常高画质（修改渲染分辨率为 1.0）星槎海中枢 55 ～ 60FPS
CS2：900P 全低（没开动态分辨率），平均大概 100 ～ 110FPS
战地 1：1080P 低画质预设，我屏幕 60Hz 的，开了垂直同步，反正能稳 60FPS（但后面玩着玩着驱动报错，就没玩了）
战地 5：1080P 低画质预设，同样的，能稳 60FPS
战地 2042：1080P 低画质预设，这个就得开动态分辨率了，不过开了后会糊成一坨，不开大部分场景又只有 30-40FPS，卡的没法子玩
赛博朋克 2077：1080P 低画质 + FSR 平衡档，跑测试场景，平均 60FPS 左右
黑神话：悟空：1920*1200 低画质 + 超采样 59，跑测试平均 69FPS

硬伤

不带最近小主机很流行的 Oculink 接口，外接显卡很麻烦，需要舍弃一个硬盘位，并手动从底部引线出来，所以对于我来说就是没得折腾……

同时这个还不支持内存超频，核显就把内存当显存用，性能瓶颈很大部分都在内存频率，而小主机用的是笔记本的 D5 条子，延迟带宽方面都是硬伤……

总结

富裕家庭可以把这玩意当作机顶盒/NAS/软路由来耍，也可以应付普通办公需求，打游戏跑 AI 嘛，还是别难为 780M 核显了，真干活还得是独显

碎碎念

点击显示/隐藏怨气碎碎念

所以最后我还是把它二手出了，出给个浙江的家伙，好心好意给他把系统重装了，要求软件下好破解好，Office 全套装好一分钱也没加……

结果给我来个不接快递电话，我联系快递员放给他丰巢，又说显示器还没到不能验货丰巢放三天先 😅，最后又说借了显示器给我验货完，又用祖宗口吻问我为什么要设置密码（你鬼问微软啊），设置了还不告诉他，最后还 TM 给我拖到自动确认收货 😅，要不是那时候正值国庆不想坏心情……
