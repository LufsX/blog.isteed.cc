安装 Magisk

⚠️ 注意：下文默认电脑已具有正确的 ADB 环境，具体配置教程可看少数派文章 —— 如何从零开始使用 adb

解锁 Bootloader

折腾这些的大前提便是解锁 Bootloader，所以本教程并不适用于无法解锁 Bootloader 的机子哦

这里就简单说一下解锁步骤

  1. 手机：开发者选项 - OEM 解锁 - 打开 - 重启进入 fastboot 模式（电源键 + 音量减）
  2. 电脑：连接手机 - 执行 fastboot flashing unlock
  3. 手机：按音量键直到看见 Unlock the bootloader - 按电源键确认
⚠️ 注意：解锁会抹除所有数据，请一定确认备份好重要数据后再解锁

获取待修补的 Boot 镜像

有两种方法～

点击展开第一种方法（推荐）

从这里找到与手机中的 设置 -> 关于手机 -> 版本号 （在最下面）相同文件名的文件，下载到手机即可

如果不存在对应文件，请使用第二种方法

接下来打开 Magisk Manager - 安装 - 选择并修补一个文件 - 选择刚刚下载的文件 - 开始

至此，修补后的 boot 镜像就做好了，将它传到电脑上备用即可


点击展开第二种方法（较繁琐）

从 Google 官方的工厂镜像下载对应机型与版本号的工厂镜像，版本号可以在 设置 - 关于手机 - Android 版本 - 版本号 中看到

接着将下载下来的工厂镜像解压，得到形如下面的文件

把里面的 boot.img 传到手机上，用 Magisk Manager 选择并修补对应文件

Magisk Manager - 安装 - 选择并修补一个文件 - 选择传到手机上的文件 - 开始

至此，修补后的 boot 镜像就做好了，将它传到电脑上备用即可


安装

  1. 手机进入 fastboot 模式（手机开机时按住 电源键 + 音量减 或电脑命令行键入 adb reboot bootloader 并回车）

  2. 电脑连接手机，执行 fastboot flash boot /path/to/boot.img
    其中，/path/to/boot.img 为你修补后的镜像位置

  3. 电脑命令执行结束后重启即可（长按电源键或电脑命令行键入 fastboot reboot 并回车）

  4. Enjoy!
